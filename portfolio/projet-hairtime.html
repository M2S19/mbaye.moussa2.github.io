Application web de gestion de rendez-vous pour les salons de coiffure

Présentation du projet

Pour mon projet de l'épreuve E5 du BTS SIO, j'ai développé une application web de gestion de rendez-vous pour les salons de coiffure. Cette solution complète permet de digitaliser la prise de rendez-vous et d'optimiser la gestion quotidienne des salons.

Fonctionnalités

  • Création de compte pour les coiffeurs
  • Ajout et gestion de salon
  • Définition des créneaux de travail
  • Création de compte pour les clients
  • Consultation des différents salons de coiffure disponibles
  • Prise de rendez-vous en ligne
